# Flashcard App

This project is a mobile Android application that allows users to study using flashcards. Each card contains a question and an answer, and users can flip the card to reveal the answer. The app also allows navigation between multiple flashcards and remembers the last viewed state using local storage.

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Open the project in Android Studio
2. Allow Gradle to sync dependencies
3. Connect an Android device or start an emulator
4. Click the "Run" button to launch the app

Instructions for using the software:

1. Press the "Flip" button to switch between question and answer
2. Press the "Next" button to navigate to the next flashcard
3. Close and reopen the app to verify that the last state is saved

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Android Studio (latest version)
* Kotlin programming language
* Jetpack Compose (UI framework)
* Material 3 components
* SharedPreferences (for local storage)
